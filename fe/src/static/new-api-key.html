Create an API Key

An API key can be a secret stored on the server or a publishable key included in your website for your users. The API key can either be a request parameter or an http header.

https://api.indexsupply.net/query?api-key=XXX&...

IP Address Connection Limits

Your account allows for {{plan.connections}} connections. You can further restrict this number to a fixed number of connections per client IP address.

If you leave this value blank or set it to 0 then any given client can consume all {{plan.connections}} connections.

Origin Domains

Optionally provide a comma separated list of domain names. i.e. foo.com, bar.com

When domain names are provided, HTTP requests using this API Key will be valid when the Host header is contained in this list of domains.

This isn't perfect security. It is possible for the Host header to be spoofed outside of a browser. But it does prevent someone from using this API Key on someone else's domain.
